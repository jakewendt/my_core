= My Core Rails App


== Expectations and Requirements

* ruby 1.8.7
* gems
  * ryanb-acts-as-list
  * prawn
  * gravatar
* gems (for testing)
  * ZenTest
  * autotest-rails


== Installation

 sudo gem install rails -v=2.3.2
 sudo rake gems:install


== Testing

 sudo rake gems:install RAILS_ENV=test
